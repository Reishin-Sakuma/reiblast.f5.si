# AIがFXの売買タイミングを教えてくれるアプリ「Charin」を作りました

<u>「AIと一緒に、FXを始めてみよう。」</u>

どうも、レイと申します。

まあ沖縄は冬でも20℃前後あったりするんで、年中そんな感じなんですけどね笑

さて、今回は僕が個人開発したFX系のアプリ「**Charin（チャリン）**」を紹介しようと思います。

ちなみに名前の由来は、**Chart（チャート）** と **Ringtone（通知・教えてくれる）** を組み合わせた造語です。「相場をチャートで見て、タイミングを通知で知らせる」というコンセプトをそのまま名前にしました。

## Charin ってどんなアプリ？

一言でいうと、**ドル円（USD/JPY）のリアルタイム相場を見ながら、AIが「買い・売り・様子見」を判断して教えてくれるPWAアプリ**です。

LP も作ったので見てみてください👇

アプリ本体はこちら↓

[https://app.charin.reiblast.f5.si](https://app.charin.reiblast.f5.si)

現在は**ベータ版**として公開中です。**完全無料・会員登録なし・インストール不要**で、URLにアクセスするだけで使い始められます。

今後は Google ログインによる会員登録機能を導入して、予測履歴のクラウド保存なども検討しています。

## 作ったきっかけ

元々ちょっとFXに興味があって、ドル円のチャートをぼーっと眺めることがあったんですよ。

ただ、テクニカル指標とかチャートの読み方とか、正直わかるようで全然わからなくて、「今、買いなの？売りなの？」って毎回迷ってたんですよね(´・ω・｀)

「これAIに聞いたらいいじゃん」って思いついたのが始まりです。

で、せっかく作るなら<span style="color:#ff7f00">**本格的なテクニカル分析も自分で実装してみよう**</span>と思って、RSI・MACD・ボリンジャーバンド・ATRまで自前で組み込む羽目になりました。完全に想定外の方向に転がりましたが、それはそれで勉強になりましたwww

## 主な機能

### リアルタイムレート表示

GMO Coin の無料 WebSocket API（認証不要）を使って、ドル円のBID/ASKをミリ秒単位でリアルタイム更新します。
チャートはSVGで自前描画しており、SMA(5)・SMA(20) のオーバーレイも付いています。

### テクニカル指標

以下の指標を全部フロントエンドで計算しています。

| 指標 | パラメータ |
|---|---|
| RSI | 14期間（Wilder EMA方式） |
| MACD | 12, 26, 9 |
| SMA | 5期間 / 20期間 |
| ボリンジャーバンド | 20期間・2σ |
| ATR | 14期間（close-to-close近似） |

MAクロス（ゴールデン/デッドクロス）を検出したら10分間ハイライト表示もします。

### AI相場分析

メインの目玉機能です。

ボタンを1回押すだけで、<span style="color:#8f20ff">Cloudflare Workers AI の Llama 3.3 70B</span> が相場を分析して「buy / sell / neutral」を返してきます。

AIに渡すデータは以下のとおり。

- 直近20件の価格推移
- RSI・MACD・SMA・ボリンジャーバンド・ATR の現在値
- 現在時刻（JST・セッション名）
- 過去10件の予測実績（方向・根拠・的中結果）

これだけ詰め込んで、AIが <span style="color:#ff007d">**「①トレンド方向 ②RSI/MACD評価 ③エントリー根拠」の3点**</span> を80字以内で説明してくれます。

buy / sell の場合はエントリー期限・決済目安（分）・目標レートも一緒に返ってきます。

### AI予測ラインのチャート可視化

AI分析の結果として出たエントリー価格・目標レート・未来ゾーンを、そのままSVGチャートの上に描画します。
テキストだけじゃなくて視覚的に確認できるのは地味に便利だなと思ってます。

### 参考損益表示

ロット数を入力すると、AI予測の目標レートまで到達したときの参考損益を自動計算してくれます。
DMM FX 基準の 1lot = 10,000通貨 で計算しています。

### 決済カウントダウン

AIが「15分後に決済」と言ったら、目標時刻まで秒単位でカウントダウンします。
時刻が来たら「目標到達 / 方向一致 / 外れ」を自動判定して表示してくれます。

### 的中率

- **手動分析**（ボタン押下）と**1時間ごとの自動予測**（Cloudflare Worker + Cron）の両方を集計
- スコア制（完全的中 1.0 / 部分的中 0.5 / 外れ 0.0）で計算
- AIを盲信しないために、実績データで判断できるようにしました

### ブラウザ通知・ダークモード・PWA

ServiceWorker 経由でブラウザ通知（決済2分前）、ダークモード対応、ホーム画面追加も対応済みです。

## 技術スタック

| 種別 | 採用技術 |
|---|---|
| フレームワーク | <strong>Vue 3</strong>（Composition API / `<script setup>`） |
| ビルドツール | Vite 7 |
| スタイル | Pico.css v2（クラスレスCSS） |
| PWA | vite-plugin-pwa + Workbox |
| ホスティング | Cloudflare Pages |
| AI分析 | <strong>Cloudflare Workers AI（Llama 3.3 70B）</strong> |
| 自動予測・的中率 | Cloudflare Worker + KV |
| データ（リアルタイム） | GMO Coin WebSocket API（完全無料・認証不要） |
| データ（履歴） | Yahoo Finance API（USDJPY=X, 1分足 2時間分） |

フロントエンドは Vue 3 + Vite でサクッと作りつつ、バックエンドの AI 呼び出しは Cloudflare Pages Functions（`/api/analyze`）に乗せています。

Cloudflare Workers AI は**無料枠で 10,000 neurons / 日**使えます。Llama 3.3 70B fp8 で1回あたり約 300〜500 neurons なので、1日20〜30回は分析できる計算ですね。

データソースの GMO Coin WebSocket は<span style="color:#0000ff">**完全無料・認証不要・レート制限なし**</span>なのが超ありがたかったです。FXのリアルタイムデータってお金かかるイメージあったんですが、こんな形で使えるとは思ってませんでした。

## こだわったポイント

### テクニカル指標を全部自前実装した

正直、チャートライブラリ使えばよかったんですが、RSIのWilder EMA方式とかMACDのヒストグラム計算とかATRとか、全部 `utils/technicalIndicators.js` に書きました。

計算式を調べながら実装したのでちゃんと動くか不安でしたが、実際のチャートと見比べてほぼ一致したのでよかったです笑

### AIに過去の予測実績を食わせた

最初は「現在の指標だけAIに渡す」設計だったんですが、過去10件の予測実績（予測方向・根拠・的中結果）もプロンプトに含めるようにしました。

これによって、<span style="color:#ff7f00">**「さっきbuyと言ったのに全然上がらなかった → 今はsellかneutralのほうが妥当では」**</span>みたいな文脈をAIが考慮できるようになります。

### 的中率をごまかさない設計にした

完全に当たった場合と「方向は合ってたけど目標未達」の場合でスコアを分けています。

AIを「使えるツール」として付き合うためには、実績を正直に記録することが大事かなと思って。
