# 24歳から見た「Geminiの世界」とは

<u>「検索の会社が作ったAIと、検索専門のAIは、どう違うんですか。」</u>

どうも、レイと申します。

昨日、Perplexityの記事の締めでこんなことを書きました。

<span style="color:#8f20ff">「あれ、Gemini書いてなくない？！ 次はGeminiを書きます。検索の会社が作ったAIが、Perplexityとどう違うのか。そのへんを整理したいと思います。」</span>

有言実行します。

沖縄、今日は昨日より晴れていい感じでした。窓開けながら記事書いてます。2月後半の沖縄、普通に快適すぎる。

---

昨日のPerplexity記事を書いていて「あれ、Gemini書いてないじゃん」と気づいたときの衝撃、自分でびっくりしました。

ChatGPT・Claudeと並ぶ代表格なのに、Perplexityを書き終えるまで完全にスルーしてた。www

でも実はこれ、理由があって。

## Bard時代に触ってイマイチだったから、ずっとスルーしてた

今の「Gemini」という名前になる前は「**[Bard](https://ja.wikipedia.org/wiki/Bard_(chatbot))**」という名前でした。2023年にリリースされて、当時ちょっと触ってみたんですが。

<span style="color:#8f20ff">「なんか、微妙じゃない？」</span>

というのが当時の感想でした。

ChatGPTのほうが使いやすかったので、そのままそっとブラウザのタブを閉じてしまった。以来ずっと「Googleのやつは一回触ったからいいかな」という感じで放置していたわけです。

それが2024年に「Gemini」というブランドに刷新されて、モデルも大幅に強化された。Perplexityの記事を書いていて「あ、ちゃんと使ってみてないじゃん」となったのが今回のきっかけです。

最初の印象で判断し続けるのよくないですね。Copilotのときもそれやったな... (^o^)

## そもそもGeminiって？

「このブログの通例なんで書きます。」（このくだり、何回目かわからなくなってきた。笑）

Geminiとは、<strong>[Google DeepMind](https://ja.wikipedia.org/wiki/Google_DeepMind)</strong>というGoogleのAI研究部門が開発した生成AIです。

大きな特徴のひとつが「**マルチモーダル**」対応で、**テキスト・画像・音声・動画・コードを組み合わせて処理できる**設計になっています。「文章しか扱えない」という制限がそもそもない。

モデルの種類は：

- **Gemini Flash** - 速い・軽い・無料でも使いやすい
- **Gemini Pro** - バランス型
- **Gemini Advanced** - 最上位。有料（Google One AI Premiumプランが必要）

という感じです。

## で、昨日書いたPerplexityと何が違うの？（ここが本題）

予告した通り、ここをちゃんと整理します。

「Geminiも検索できる」「Perplexityも検索できる」。同じに見えるんですが、**根本的なコンセプトが全然違います**。

| | Gemini | Perplexity |
|---|---|---|
| 何者か | 生成AIに検索機能を足した | 検索エンジンをAIで強化した |
| 主な得意技 | 生成・相談・会話 | 調べる・引用する・まとめる |
| 出典の明示 | 場合による | 常に明示 |
| コード生成 | できる | 苦手 |
| 文章を一から作る | 得意 | やや物足りない |
| Googleとの連携 | 強い | なし |

一言で言うと、<strong>「Perplexityは調べることが目的、Geminiは作ることが目的」</strong>です。

Perplexityは「どこに書いてあるか」を見つけるのが使命で、出典を必ず出してくれます。GeminiはAIとして文章を生成したり相談に乗ったりすることが主軸で、そのついでに最新情報も引っ張れる。同じ「Web検索できるAI」でも、使う局面が違うんです。

<span style="color:#8f20ff">「結局どっちを使えばいいの？」</span>

という話になりますが、**調べた結果がほしいならPerplexity、AIに作業を手伝ってもらいながら最新情報も必要ならGemini**というのが今の自分の答えです。

## ChatGPTとの違いは「Googleの強みをそのまま持ち込んでいる」

Perplexityとの比較はしたので、ChatGPTとの違いも書きます。

Geminiの差別化は、<strong>「GoogleであることをフルMAXで活かしてくる」</strong>という点です。

**リアルタイム検索**

ChatGPTには学習データに期限があって、最新情報に弱い話を前の記事で書きました。Geminiは**Googleの検索エンジンと連携していて、最新の情報をリアルタイムで拾って回答できます**。最新のパッチ情報やCVEの詳細を調べるとき、この差が出ます。

**Google Workspaceとの連携**

<strong>[Gmail](https://ja.wikipedia.org/wiki/Gmail)</strong>や<strong>[Google ドキュメント](https://ja.wikipedia.org/wiki/Google_%E3%83%89%E3%82%AD%E3%83%A5%E3%83%A1%E3%83%B3%E3%83%88)</strong>、<strong>[Google スプレッドシート](https://ja.wikipedia.org/wiki/Google_%E3%82%B9%E3%83%97%E3%83%AC%E3%83%83%E3%83%89%E3%82%B7%E3%83%BC%E3%83%88)</strong>との連携があって、「このメールに返信して」「この書類を要約して」みたいなことができます。業務でGoogleを使っている組織には刺さると思う。

**コンテキスト長**

Claudeのときも話しましたが、Geminiも上位モデルは**100万トークン以上**のコンテキストに対応しています。大量のログをそのまま全部投げる、みたいな雑な使い方ができるのはインフラエンジニア的にありがたい。

## 実際に使ってみた正直な話

**回答の品質そのものは十分高い**です。ハルシネーションの頻度もChatGPTよりは抑えられてる印象があって、「それっぽいけど実は嘘」みたいなパターンは少ない気がします。気がします、レベルだけど。

<span style="color:#8f20ff">「じゃあなんでClaudeをメインで使ってるの？」</span>

それはですね、**「汲む」精度がClaudeより弱い気がする**んですよね。

Claudeは「これが聞きたかったんでしょ」という感じで返してくれるんですが、Geminiはたまに的がずれた方向で丁寧に答えてくる。答えは正確なんだけど「そっちじゃなかった」ってなる。うまく言えないんですが、「会話のテンポ感」が自分に合わなかった、というか。

---

一方で「**これはGeminiじゃないとダメだな**」という場面も確かにありました。

最新技術調査はやはり強い。「〇〇の最新バージョンで追加された機能を教えて」みたいな質問は、Geminiのリアルタイム検索が一番速くて正確でした。ここはClaude・ChatGPTに対して明確に優位に感じる部分です。

あとはGoogle Workspaceのドキュメント整理。スプレッドシートのデータを貼り付けてサマリーを出す使い方は普通に便利でした。Google同士の親和性はさすが、という感じ。

## 気になるところも正直に書く

**一番気になるのは、Googleへの依存が前提になること**です。Geminiの強みはGoogle Workspaceとの連携ですが、それって「Googleを使い続けることが前提」でもあります。プライバシーの観点で「Googleに全部持たせていいのか」という意識がある人もいるだろうし、Google Workspaceを使っていない組織では強みが半減する。

<span style="color:#ff0000">**コーディング補助はClaudeに一歩劣る印象**</span>があります。コードを生成・デバッグする場面でClaudeに慣れた身からするとちょっと物足りない。悪くはないんですが、「わざわざ切り替えるほどか？」となってしまう。

あとは料金。Gemini AdvancedはGoogle One AI Premiumプランが必要で月額2,900円。Claude Proの月$20と競合するレンジですが、片方だけ選べと言われたら今は迷わずClaudeです。

<span style="color:#8f20ff">正直に言いすぎてGoogleに怒られそう。笑</span>

## これで5本書いてきたので全部整理する

ChatGPT・GitHub Copilot・Claude・Perplexity・Geminiと書いてきたので、改めてポジションをまとめます。

- <span style="color:#0000ff">**ChatGPT**</span> → 生成AIへの入口として最高。なんでも相談できる汎用的な友達
- <span style="color:#0000ff">**GitHub Copilot**</span> → エディタに常駐するコード補完特化のアシスタント
- <span style="color:#0000ff">**Claude**</span> → 長文・深い相談向き。今の自分のメイン
- <span style="color:#0000ff">**Perplexity**</span> → リアルタイムで調べて、出典付きで教えてくれる情報収集特化型
- <span style="color:#0000ff">**Gemini**</span> → GoogleのエコシステムとAIを組み合わせたい場面に強い

<span style="color:orange">**どれかひとつが「最強」なんじゃなくて、場面ごとに最強が変わる。**</span>

これが5本書いてきた結論です。ひとつのツールに全部任せようとすると、どこかで限界が来ます。
