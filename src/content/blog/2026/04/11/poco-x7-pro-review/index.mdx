## 買った経緯

もともと [Redmi Note 10T](https://ja.wikipedia.org/wiki/Redmi_Note_10) を使ってたんですよね。Amazon で SIMフリー版を買ったやつ。普段使いには不自由しないけど、ゲームやるとカクつくし、カメラもまあ…値段なりだなって感じ。[Snapdragon](https://ja.wikipedia.org/wiki/Qualcomm_Snapdragon) 480 5G に 4GB RAM という構成だったので、最近のアプリにはちょっと力不足になってきてました。

特にキツかったのがアプリの切り替え。4GB RAM だと裏で落ちまくるんですよね。ブラウザ開いて → LINE 返して → ブラウザ戻ったらリロードかかる、みたいな。あとゲームの読み込みも遅いし、[ドリスピ](https://ja.wikipedia.org/wiki/%E3%83%89%E3%83%AA%E3%83%95%E3%83%88%E3%82%B9%E3%83%94%E3%83%AA%E3%83%83%E3%83%84) はまあ動くけど、レーシングマスター はカックカクで正直まともに遊べなかった（´・ω・｀）

そろそろ乗り換えたいけどハイエンドは10万超えが当たり前の時代だし…って悩んでたときに見つけたのが [POCO](https://ja.wikipedia.org/wiki/POCO_(%E3%83%96%E3%83%A9%E3%83%B3%E3%83%89)) X7 Pro。[Xiaomi](https://ja.wikipedia.org/wiki/%E3%82%B7%E3%83%A3%E3%82%AA%E3%83%9F) のサブブランドである POCO から出ているミッドハイモデルで、**4万円台で [Dimensity](https://ja.wikipedia.org/wiki/MediaTek_Dimensity) 8400-Ultra 搭載**という頭のおかしいコスパ。これは試すしかないでしょと。楽天でポチりました。

## パフォーマンス

### 普段使い

はっきり言って**別世界**です。Redmi Note 10T の Snapdragon 480 + 4GB RAM から、Dimensity 8400-Ultra + 12GB RAM に変わったので当然っちゃ当然なんですけど。

アプリの切り替えがサクサク。ブラウザのタブを20個くらい開いても全然もたつかないし、SNSアプリを何個も同時に開いてても裏で落ちない。Redmi Note 10T 時代の「ブラウザ戻ったらリロード地獄」から完全に解放されました（笑）

[UFS](https://en.wikipedia.org/wiki/Universal_Flash_Storage) 4.0 ストレージの恩恵もデカい。Redmi Note 10T は eMMC 5.1 だったので、アプリのインストールや起動速度が**体感で3倍くらい**違います。マジで。

### ゲーム

ここが一番変化を実感したところ。僕は [レースゲーム](https://ja.wikipedia.org/wiki/%E3%83%AC%E3%83%BC%E3%82%B9%E3%82%B2%E3%83%BC%E3%83%A0) が好きで [ドリスピ](https://ja.wikipedia.org/wiki/%E3%83%89%E3%83%AA%E3%83%95%E3%83%88%E3%82%B9%E3%83%94%E3%83%AA%E3%83%83%E3%83%84) と レーシングマスター をよくやるんですけど、Redmi Note 10T だとレーシングマスターがカックカクでまともに遊べなかったのが、POCO X7 Pro だと**最高画質でヌルヌル**。マジで同じゲームか？ってレベルで変わりました。ドリスピも当然サクサク。

120Hz [AMOLED](https://ja.wikipedia.org/wiki/%E6%9C%89%E6%A9%9F%E3%82%A8%E3%83%AC%E3%82%AF%E3%83%88%E3%83%AD%E3%83%AB%E3%83%9F%E3%83%8D%E3%83%83%E3%82%BB%E3%83%B3%E3%82%B9) ディスプレイとの組み合わせが最高で、スクロールもゲームもとにかく滑らか。Redmi Note 10T の IPS 90Hz から乗り換えると、画面の綺麗さと滑らかさの差に感動します。

{/* TODO: 写真 gaming.jpg — レーシングマスターのプレイ画面 */}

長時間プレイしても発熱は控えめ。30分くらいレーシングマスター回してもほんのり温かくなる程度。Redmi Note 10T はゲーム中にけっこう熱くなってたので、この差は大きい。

## カメラ

メインカメラは **50MP の [Sony](https://ja.wikipedia.org/wiki/%E3%82%BD%E3%83%8B%E3%83%BC) IMX882** で、OIS（光学式手ブレ補正）対応。Redmi Note 10T の Samsung JN1 センサーからの乗り換えですが、**センサーの差より OIS の有無がデカい**。手ブレでボケる写真が激減しました。

{/* TODO: 写真 camera-daylight.jpg — POCO X7 Pro で日中に撮った写真 */}

日中の撮影は文句なし。色の再現性も自然で、Redmi Note 10T だと白飛びしがちだったシーンでもしっかり撮れます。HDR処理が優秀。

{/* TODO: 写真 camera-night.jpg — POCO X7 Pro で夜景モードで撮った写真 */}

夜景モードもそこそこ使えます。Redmi Note 10T の夜景はノイズまみれで正直使い物にならなかったんですけど、POCO X7 Pro はちゃんと見れるレベルで撮れる。ハイエンドの Pixel や iPhone と比べるとさすがに差はありますが、「4万円台のスマホで撮ったの？」ってレベルには達してます。

超広角カメラは8MPなので、こっちはおまけ程度。まあ風景撮るときにたまに使う程度なんで、メインカメラがしっかりしてれば個人的にはOKです。

## バッテリー・充電

**6,000mAh** の大容量バッテリー。Redmi Note 10T の 5,000mAh でも「バッテリー持ちいいな」って思ってたんですけど、POCO X7 Pro はさらに上をいきます。

普段使いなら**1日半〜2日**は余裕。朝100%で出発して、SNS見たりブラウジングしたり音楽聴いたりして、夜帰ってきてもまだ40%くらい残ってることが多い。

そして最大の感動ポイントが **90W急速充電**。Redmi Note 10T は 18W だったので、0%からフル充電まで2時間以上かかってたんですよね。POCO X7 Pro は **約45分** で満充電。朝起きてシャワー浴びてる間に充電すれば、出かける頃にはほぼフルになってます。これに慣れると18Wにはもう戻れない…。

ちなみに僕は [前回レビューした UGREEN の100W充電器](/blog/ugreen-100w-charger-review) で充電してますが、相性抜群です(^o^)

## 気になった点

正直に書きます。

<strong>Xiaomi HyperOS のクセが強い。</strong>
[Android](https://ja.wikipedia.org/wiki/Android_(%E3%82%AA%E3%83%9A%E3%83%AC%E3%83%BC%E3%83%86%E3%82%A3%E3%83%B3%E3%82%B0%E3%82%B7%E3%82%B9%E3%83%86%E3%83%A0%29)) ベースとはいえ、[HyperOS](https://ja.wikipedia.org/wiki/Xiaomi_HyperOS) は独自カスタマイズが結構入ってます。Redmi Note 10T は [MIUI](https://ja.wikipedia.org/wiki/MIUI) だったので Xiaomi の UI には慣れてましたが、HyperOS になって通知の挙動が微妙に変わったりして最初は戸惑いました。慣れれば問題ないんですけど、Pixel や素の Android から来た人は結構面食らうかも。あと、プリインストールアプリが多め。使わないやつは速攻で消しました笑

<strong>おサイフケータイ / FeliCa 非対応。</strong>
これが Redmi Note 10T からの**唯一のダウングレード**。Redmi Note 10T は [FeliCa](https://ja.wikipedia.org/wiki/FeliCa) 搭載で [Suica](https://ja.wikipedia.org/wiki/Suica) も [おサイフケータイ](https://ja.wikipedia.org/wiki/%E3%81%8A%E3%82%B5%E3%82%A4%E3%83%95%E3%82%B1%E3%83%BC%E3%82%BF%E3%82%A4) も使えてたんですよね。POCO はグローバルモデルなので FeliCa なし。沖縄だとそもそも電車がないので僕はあんまり困ってないんですけど（笑）、都市部で Suica をガンガン使ってる人にとっては致命的かもしれないです。[Visa タッチ決済](https://ja.wikipedia.org/wiki/Visa%E3%81%AE%E3%82%BF%E3%83%83%E3%83%81%E6%B1%BA%E6%B8%88) や [Google Pay](https://ja.wikipedia.org/wiki/Google_Pay) の NFC 決済で代用はできます。

**スピーカーの音質はそこそこ。**
ステレオスピーカー搭載ですが、ハイエンドと比べると低音が弱め。ただ Redmi Note 10T のモノラルスピーカーよりは圧倒的に良いです。動画視聴くらいなら全然問題ないですけど、スピーカーで音楽をガッツリ聴くタイプの人は物足りないかも。僕はイヤホン派なのであんまり気にならないですけどね。
