# ブログ公開ツイート テンプレート

記事を公開したときのXポスト用テンプレート集。
記事の種類・内容に合わせて選んで使う。

---

## 基本テンプレート

```
ブログ書きました！

📝【記事タイトル】

〈一言で内容を説明〉

🔗 https://reiblast1123.com/blog/YYYY/MM/DD/slug/

#エンジニアブログ #インフラエンジニア #タグ
```

---

## 技術系・インフラ系

```
ブログ書きました📝

【記事タイトル】

〈検証・調査した内容を一言で〉
同じ状況で詰まってる人の参考になれば！

🔗 https://reiblast1123.com/blog/YYYY/MM/DD/slug/

#インフラエンジニア #エンジニアブログ #タグ
```

---

## 趣味・日常系（クルマ・ゲームなど）

```
ブログ書いたよ〜

【記事タイトル】

〈内容をカジュアルに一言〉
よかったら読んでみてください(^o^)

🔗 https://reiblast1123.com/blog/YYYY/MM/DD/slug/

#タグ #タグ
```

---

## Ameba 移植記事（過去記事の再掲）

```
昔Amebaに書いたやつをブログに移植しました

【記事タイトル】

〈当時の状況・内容を一言〉
黒歴史感あるけどせっかくなので笑

🔗 https://reiblast1123.com/blog/YYYY/MM/DD/slug/

#エンジニアブログ #タグ
```

## よく使うハッシュタグ候補

| カテゴリ | ハッシュタグ |
|---|---|
| 共通 | `#エンジニアブログ` `#ブログ書いた` |
| 職業 | `#インフラエンジニア` `#SRE` `#ネットワークエンジニア` |
| 技術 | `#Linux` `#Network` `#Cloudflare` `#AWS` `#Nutanix` |
| クルマ | `#車好きと繋がりたい` `#車検` `#ドライブ` |
| ゲーム | `#レースゲーム` `#グランツーリスモ` `#アセットコルサ` |
| 沖縄 | `#沖縄` `#沖縄エンジニア` |

---

## 文字数目安

- 日本語1文字 ≒ 2カウント（X は280文字制限）
- URL は 23 カウント固定
- ハッシュタグ含めて **実質 100〜120 文字以内** を目安にするとゆとりが出る
